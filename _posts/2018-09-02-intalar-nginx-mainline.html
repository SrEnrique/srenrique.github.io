Actualizar repositorios de Ubuntu

Antes de iniciar tenemos que actualizar los repositorios de Ubuntu

$ sudo apt update && sudo apt dist-upgrade

Agregar repositorio de nginx

Agregamos los repositorios de la última versión estable de nginx

$ cd /tmp/ && wget http://nginx.org/keys/nginx_signing.key
$ sudo apt-key add nginx_signing.key

cuando esto termine agregamos el repositorio a la lista sources.list.d

$ sudo sh -c "echo 'deb http://nginx.org/packages/mainline/ubuntu/ '$(lsb_release -cs)' nginx' > /etc/apt/sources.list.d/nginx.list"

Intalar Nginx


$ sudo apt update
$ sudo apt install nginx
